## Synchronize Ewon Flexy Tag Data to Canary's Tag Historian
+ Wide range of Ewon Flexy IO servers for various PLC connections
+ Flexible tag configurations to tune historical data, alarming, and more
+ Canary's high performance tag writes allows multiple connectors to send to a single historian if desired

## Secure and Effective
+ Makes use of the Canary Sender API (v2)
+ HTTPS Authentication using Certificates
