<>The {ScDocusaurusConfig.title} connector can be downloaded by clicking the 'Download' button in the menu above (top-left
corner for mobile and small devices).</><br/><br/>

To run and install the connector, you need to download the latest asset named `flexy-canary-connector-A.B.C-full.jar` where A.B.C
are the Canary version numbers, the jvmrun file, and the CanaryConnectorConfig.json

:::note
The CanaryConnectorConfig will need to be modified to appropriate values described [here](setup/usage-configuration)
before uploading to the device.
:::
